angularjs-typescript
====================

This repository is planned as bootstrap for AngularJS projects

Installation
------------

You need grunt-cli, npm, hoodie-cli and bower in the first place.

```cd config ```

```npm install ```

```bower install ```


Commands
--------
Development:

```grunt watch ```
```hoodie start -www "../build"```

Production:

```grunt ```
```hoodie start -www "../www"```
